AquaMate

AquaMate is a unified hub for aquatic life and aquarium care — a place for hobbyists of any experience level to browse and learn about a wide range of fauna and flora, and to build mock aquariums for planning. It began as a San Francisco State University CS capstone (forked from the department's project under Professor Song) and is live at aquamate.me.

Architecture

  • Frontend: React + React Router single-page app, with a custom species browser, search, and an interactive mock-aquarium builder tool.
  • Backend: a Cloudflare Worker API serving species data and imagery.
  • Storage: Cloudflare R2 buckets for fauna and flora image assets.
  • Data pipeline: tooling to compile, migrate, and validate the species schema that powers the catalog.

Features

  • Species catalog — browse and search a large array of aquatic fauna and flora with care information.
  • Aquarium builder — assemble mock tanks for planning, mixing species and layout.
  • Education-first — approachable info pages aimed at hobbyists of every experience level.
